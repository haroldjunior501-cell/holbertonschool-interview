# N Queens

Solve the N queens puzzle using backtracking.

## Description

Place N non-attacking queens on an N×N chessboard and print every solution.

## Usage

```bash
./0-nqueens.py N
```

- `N` must be an integer greater than or equal to 4
- One solution per line
- Format: `[[row, col], [row, col], ...]`

## Files

| File | Description |
|------|-------------|
| `0-nqueens.py` | Solves and prints all N queens solutions |

## Example

```bash
./0-nqueens.py 4
[[0, 1], [1, 3], [2, 0], [3, 2]]
[[0, 2], [1, 0], [2, 3], [3, 1]]
```
