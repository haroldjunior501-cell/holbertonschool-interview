# UTF-8 Validation

Determine if a list of integers represents a valid UTF-8 encoding.

## Description

Each integer represents one byte (only the 8 least significant bits are used).
A UTF-8 character can be 1 to 4 bytes long.

## Files

| File | Description |
|------|-------------|
| `0-validate_utf8.py` | Returns `True` if data is valid UTF-8, else `False` |

## Prototype

```python
def validUTF8(data)
```
